# CelestiaAddons

A comprehensive Minecraft Fabric mod for version 1.21.10 that brings powerful automation and visual enhancements to your gameplay.

## Features

### AutoRoute System
- **Safe Mode**: Watchdog-safe movement using OnGround packets
- **Fast Mode**: High-speed movement for maximum efficiency
- **Noamm-Style Visuals**: 3D boxes, lines, and markers for route visualization
- **DungeonBreaker Integration**: Automatic block breaking during autoroutes

### ESP Features
- **Player ESP**: 3D boxes and tracers for player detection
- **Chest ESP**: 3D boxes highlighting chests and containers
- **Star Mob ESP**: Enhanced mob detection (inherited from NoammAddons)

### Utility Features
- **Fullbright**: Gamma adjustment for maximum brightness
- **Speed**: Movement speed multiplier
- **DungeonBreaker**: Automatic block breaking in configurable radius

### Technical Features
- **Mixin Support**: Enhanced client-side modifications
- **Watchdog Safety**: Designed to avoid detection
- **Configurable Settings**: Extensive customization options
- **Clean GUI**: Intuitive interface for feature management

## Installation

1. Ensure you have Minecraft 1.21.10 and Fabric Loader installed
2. Download the latest release JAR file
3. Place the JAR in your `.minecraft/mods` folder
4. Launch Minecraft with Fabric

## Building from Source

```bash
./gradlew build
```

The built JAR will be in `build/libs/`.

## Dependencies

- Fabric API
- Fabric Loader 0.16.10+
- Java 21

## Usage

- Press the designated key (configurable) to open the CelestiaAddons GUI
- Enable/disable features as needed
- Configure settings for optimal performance

## Safety Notice

This mod is designed with Watchdog safety in mind, but use at your own risk. Always follow server rules and terms of service.
